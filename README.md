### 프로젝트를 유니티 에디터에서 테스트 및 실행할 경우.  
에디터에서의 테스트 조작은 아래 플립이 **스페이스바**, 위 플립이 **B**버튼입니다.  
상세한 도움말은 게임 내 ?버튼을 클릭하세요.

+) 에디터상에서는 커스텀 맵 적용이 안됩니다.
